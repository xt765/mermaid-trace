# 安全策略

## 支持的版本

本节告知用户项目的哪些版本目前受安全更新支持。

| 版本    | 支持状态           |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |
| < 0.1   | :x:                |

## 报告漏洞

本节告知用户如何报告漏洞。

请说明去哪里报告，他们大概多久能收到关于已报告漏洞的更新，以及如果漏洞被接受或拒绝会发生什么等。

请向 [team@logcapy.dev](mailto:team@logcapy.dev) 报告漏洞。我们将在 48 小时内回复。
